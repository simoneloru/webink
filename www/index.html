Help

CrossInk running fully in your browser. Short guide — no server, no account.

What this is

This is the CrossInk e-reader firmware, compiled for the web. The panel is the same X4-style screen as the hardware simulator. Everything (firmware + your books) stays on this device / browser.

Add a book

  1. Tap Add book in the top bar (or drag an .epub onto the page).
  2. The file is copied into a virtual SD card in browser storage.
  3. On the reader, open Browse files (File browser) and pick the book under /books.

The home screen does not always refresh by itself after an import — use the on-device file browser the first time.

Where books are stored

Path inside the app: /fs_/books. On disk that is IndexedDB in this browser, not Downloads and not a cloud.

On iPhone, Safari and the Home Screen icon are separate storage. Always open from the same place you used to add books. After “Add book”, wait until the status says the library was saved before leaving the app — iOS may kill the page before a background save finishes.

Clearing site data, private mode, or low storage can erase the library. Use Library to see size, quota, and delete books.

Controls

  • Side ▲ / ▼ — page back / forward (like side keys).
  • Front ◀ ● ▶ — left / OK / right.
  • Back / Power — escape / power (mapped to the simulator).
  • Touch on the panel — edges act as arrows; centre is OK; long-press is Back.
  • Keyboard arrows, Enter, Escape, and P also work when the canvas is focused.

Pages in this web app

  • Reader — the device screen (default).
  • Library — list books, storage used, delete, add more.
  • Help — this page.

Screen orientation

Top bar icons: ↻ Rotate cycles CrossInk orientation (portrait → landscape → inverted → landscape the other way). 📱 Auto follows the phone when you turn it (default on for phones / Home Screen, off on desktop so a landscape monitor does not force landscape after a few seconds). Using Rotate turns Auto off so the choice sticks; turn Auto back on to follow the device again. ? opens this Help page. You can still change orientation from CrossInk Settings as well.

Tips & limits

  • Large EPUBs use browser storage quota (especially tight on iOS Safari).
  • Wi‑Fi / OTA / hardware transfer menus may appear but are stubbed or unused on the web.
  • You can install this site as a PWA (“Install”) for a fuller-screen experience when the browser offers it.
  • Orientation uses the browser Screen Orientation API (not the raw gyroscope). Lock screen rotation in the OS if you want to freeze it.
